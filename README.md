Die folgende Animation zeigt das Herzstück der Anwendung – die Live-Schnittstelle zu MATLAB:

Der gezeigte Prozess:

Simulation: Im Hintergrund (rechtes Fenster) wird eine Trafoberechnung in MATLAB gestartet.

Export: MATLAB speichert die Ergebnisse automatisch als CSV-Datei.

Erkennung (Watchdog): Das Python-Tool erkennt die neue Datei sofort im Hintergrund.

Import: Ein Pop-up informiert den Nutzer. Mit einem Klick werden alle technischen Daten und das korrekte Schaltbild (Vektorgrafik) geladen.

Output: Das fertige Typenschild wird per Knopfdruck als PDF generiert.
